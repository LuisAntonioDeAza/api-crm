# api-crm
## Web para administrar clientes, agregar eliminar editar y borrar usando un archivo JSON-SERVER para guardar los datos y simular una API.
##### Esta web esta echa con vite-react utilizando React Router para la navegacion entre paginas, tailwindcss para los estilos, Formik para los formulario y Yup para validar los campos de los formularios.


*Frameworks y librerias a instaladas*

[React Router](https://reactrouter.com/).

[Tailwindcss](https://tailwindcss.com/).

[Formik](https://formik.org/).

[Yup validation](https://www.npmjs.com/package/yup).


https://github.com/typicode/json-server
